# KHU_PON

## What is KHU_PON? 
- KHU is abbreviation of Kyung Hee University
- PON means coupon
- So, KHU_PON indecates KHU handouts coupon

## Perpose
- Develop for simbiosis of University and local store.

## Function
- Beacon that all over the KHU campus sends bluetooth signal to everyboy near it
- The person's smartphone that received the signal from beacon popups(actually it is push alarm) the web app service called KHU_PON
- KHU_PON, the web app service issues the coupon that could get a discount of a store near campus
